# Tatani Onboarding + Chat

Веб-приложение для поиска и предоставления помощи между людьми и волонтёрами.  
Есть два режима: «Мне нужна помощь» и «Я хочу помогать».

## Как запустить

1. Скачайте или склонируйте весь проект.
2. Откройте файл `index.html` в браузере.
3. Все функции работают локально через браузер, сервер не требуется (кроме webhook для чата AI).

---

## Возможности

- Регистрация клиента и волонтёра.
- Постановка задач с категорией, городом и геолокацией.
- Просмотр задач в кабинете волонтёра, фильтры по категориям и городу.
- Встроенный чат с AI-ассистентом (через webhook).
- Сохранение истории запросов клиента.

---

## Описание основных файлов

- **index.html** — основная разметка, точки входа.
- **/css/style.css** — стили приложения.
- **/js/** — весь JavaScript по модулям.
- **/assets/** — картинки, аватарки и т.д.
